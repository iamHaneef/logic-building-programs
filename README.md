# Logic Building Programs

## 📌 Overview

A collection of basic programming problems implemented in Java to strengthen problem-solving skills and core coding fundamentals.

---

## ⚡ Topics Covered

### Numbers & Math
- Factorial  
- Fibonacci Series  
- Largest Number  
- Prime Number  
- GCD (Greatest Common Divisor)  
- Swap Two Numbers  
- Sum of Digits  
- Permutation  

### Strings & Characters
- Reverse String  
- Count Vowels  
- Palindrome Check
